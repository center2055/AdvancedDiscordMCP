## New Features

- **Staff Logging System**: Automatic logging of all message deletions to staff logs channel
  - Logs include channel, author, message content, reason, and jump links
  - Works for both bot commands and Discord UI deletions
  - Beautiful embed format with timestamps

- **Enhanced Message Formatting**: All bot messages now use embeds by default
  - Improved visual presentation with colored embeds
  - Better readability and professional appearance
  - Customizable embed titles and colors

- **Improved Channel Organization**: Better server structure and permissions
  - Announcement channels protected from regular users
  - Welcome channel positioned at top
  - Duplicate channels and roles removed
